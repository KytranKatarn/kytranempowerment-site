Framework Details

DISA STIG — Ubuntu 22.04
51 Rules
Covers SSH configuration, password policies, audit logging, kernel hardening, file permissions, and service management. Based on the Defense Information Systems Agency Security Technical Implementation Guide.
DISA STIG — Docker
30 Rules
Covers Docker daemon configuration, container isolation policies, image security scanning, resource limits, and centralized logging. Ensures containers run with minimal attack surface.
DISA STIG — Network
27 Rules
Covers firewall rules, port exposure, TLS configuration, DNS security, and network protocol hardening. Validates defense-in-depth network architecture.
HIPAA Security Rule (164.312)
30 Rules
Covers access control mechanisms, audit controls, data integrity verification, person or entity authentication, and transmission security. Aligned with the HIPAA Security Rule technical safeguards.
CIS Benchmark — Ubuntu
40 Rules
Center for Internet Security benchmark covering filesystem configuration, software updates, process hardening, network configuration, logging, and access/authentication settings.
